# Angular App Template


## What is this?

Just my way to structure Angular applications, making this document just to remember how to do it next time.

## Folder structure (Nebular)

```
Angular Project/
|-- src/
|   |-- app/
|   |   |-- app-routing/
|   |   |   |-- app-routing.module.ts
|   |   |   |-- routes.ts
|   
|   |   |-- core/
|   |   |   |-- services/
|   |   |   |   |-- process-http-message.service.ts
|   |   |   |   |-- process-http-message.service.spec.ts
|   |   |   |-- classes/
|   |   |   |-- constants/
|   |   |   |-- functions/
|   
|   |   |-- shared/
|   |   |   |-- shared.module.ts
|
|   |   |-- app.component.ts
|   |   |-- app.component.css
|   |   |-- app.component.html
|   |   |-- app.module.ts
|   |-- assets/
|   
|-- .gitattributes
|-- .gitignore
|-- angular.json
|-- package.json
|-- package-lock.json
|-- README.md
|-- tsconfig.app.json
|-- tsconfig.json
|-- tsconfig.spec.json
```
